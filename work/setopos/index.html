The brief

Most task managers are designed for teams and loaded with features nobody asked for — integrations, kanban boards, comment threads. They end up interrupting the very thing you're trying to do: the work.

Setopos was built on one idea: the best productivity tool gets out of your way. A clean daily task list, a built-in focus timer, and nothing else. No projects, no sub-tasks, no overwhelm.

Design principles

Every UI decision was evaluated against one question: does this help the user do their work, or does it distract from it? If the answer was the latter, it didn't ship.

The interface defaults to a daily view. You see today's tasks and a focus timer. That's it. Completed items fade out. Incomplete items roll over at midnight.

Technical approach

Built with React on the frontend, Node.js + PostgreSQL on the backend, hosted on Vercel with Supabase for auth and real-time sync. Time-to-interactive under 1 second on a 4G connection.

The focus timer uses the Web Audio API to generate subtle tones at session start and end — no dependency needed, sub-3KB implementation. 

// Web Audio focus chime — zero dependencies
function playChime(freq = 440) {
  const ctx = new AudioContext();
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  osc.connect(gain);
  gain.connect(ctx.destination);
  osc.frequency.value = freq;
  gain.gain.setValueAtTime(0.3, ctx.currentTime);
  gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 1.5);
  osc.start();
  osc.stop(ctx.currentTime + 1.5);
}
